# Ruminococcaceae Analysis Agent

Multi-AI agent for Ruminococcaceae microbiome analysis.

## AI Models:
- **Claude**: Pipeline orchestration & bioinformatics
- **Gemini**: Literature review & biological interpretation  
- **ChatGPT**: Additional analysis

## Structure:
- `data/`: Input data
- `scripts/`: Agent code
- `results/`: Outputs
- `configs/`: API keys (not tracked)
- `logs/`: Execution logs
